II. Related Works

Traditional visual geolocalization often uses retrieval, classification, or hybrid pipelines. Retrieval methods compare a query image against geo-tagged references, while classification methods discretize the world into geographic cells or regions. Hybrid systems such as PIGEON combine learned geographic priors with strong visual representations.

Foundation-model approaches such as StreetCLIP and multimodal geolocalization systems add semantic cues from captions, metadata, or language-image pretraining. The original proposal expected these text signals to help by exposing geographic cues that are hard to read directly from pixels.

More recent generative methods, including diffusion and flow-matching geolocalization work, predict spatial distributions instead of only point estimates. This project fits into that line: it asks which modeling choices make probabilistic geolocalization useful in a local Swiss setting.
