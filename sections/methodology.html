III. Methodology

Deterministic Baseline Model

For the geolocalization task, the simplest baseline is a deterministic regression model, which we use as the baseline in this project. The following figure illustrates the architecture.

We first feed the input image to a pretrained image embedding model (e.g., StreetCLIP), extract the image features, and use an MLP head to output longitude and latitude for geolocalization.

This deterministic model suffers a few problems:

  • The natural human geolocalization is an uncertain task, which means the location predictions for a single image for a human could be many results. Each result has its confidence or probability to be the truth.
  • There is no quantifying metric to measure the ground-truth location confidence for a deterministic model, because in most scenarios, a user not only wants the predicted location to be more precise, but also wants the probability of the ground-truth position to be large.

Therefore, a probabilistic model is preferred in our project.

Image Conditioning

We follow Dufour et al. [8] and use a frozen StreetCLIP vision encoder [7] as the visual backbone. We experimented with both the projected StreetCLIP image embedding (768 dimensions) and the raw CLS token from the vision transformer (1024 dimensions). The CLS representation performed better and was therefore retained in the final model. We also evaluated DINOv2 as an alternative visual encoder, but did not observe improvements over StreetCLIP. The selected 1024-dimensional CLS embedding is projected to the model dimension through a LayerNorm followed by a linear layer.

Local Swiss Coordinate System

Rather than operating directly on latitude and longitude, we project all locations into a local planar coordinate system centered on Switzerland. Latitude and longitude coordinates are converted into local kilometer coordinates using a fixed reference point near the center of Switzerland and then normalized to a fixed$[-1,1]^2$ region.

This representation is a deliberate simplification. Since our task is restricted to Switzerland, Earth curvature is negligible at this scale and a local planar approximation is sufficient. This allows us to replace geodesic trajectories on the sphere with standard Euclidean flow matching while keeping the coordinate system simple and numerically stable.

The normalized Swiss bounding box defines the valid sampling region. Particles that move outside it are clipped back inside, which can occasionally create visible concentrations near the borders. Future work could replace this rectangular approximation with the actual Swiss boundary.

Flow Matching Objective

Given an image and its target location $x_{\text{target}}$, we sample a random starting point uniformly over Switzerland:

$$ x_{\text{base}} \sim \mathrm{Uniform}([-1,1]^2) $$

We then construct an intermediate point along the straight path between the two locations:

$$ x_t = (1-\gamma)x_{\text{base}} + \gamma x_{\text{target}} $$

where $\gamma$ is a random interpolation time. Rather than using ($\gamma=t$), we follow Dufour et al. [8] and apply a sigmoid scheduler ($\gamma=\kappa(t)$). This gives more weight to training points where the sampled location is already closer to the ground-truth location. According to Dufour et al., this encourages the model to focus on finer localization cues rather than only learning coarse geographic information. The target velocity is the constant displacement between the start and target locations:

$$ v_{\text{target}} = x_{\text{target}} - x_{\text{base}} $$

The model receives $(x_t,\gamma,\text{image})$ and predicts a 2D velocity field. Training minimizes the mean squared error between predicted and target velocities:

$$ \mathcal{L} = \frac{1}{2}\lVert v_{\text{pred}} - v_{\text{target}} \rVert^2 $$

Classifier-Free Guidance

Following Dufour et al. [8], we train the model with condition dropout, allowing both conditional and unconditional velocity predictions. At inference time, the guided velocity is

$$ v_{\text{guided}} = (1+\lambda)v_{\text{cond}} - \lambda v_{\text{uncond}} $$

where $\lambda$ controls the guidance strength. Our final model uses $\lambda = 0$, corresponding to purely conditional sampling.

Flow Matching Network and Inference

The trainable model is a 12-layer AdaLN MLP conditioned on the StreetCLIP embedding and the interpolation time $\gamma$. Given an image, inference starts from 64 particles sampled uniformly over Switzerland. The learned velocity field is then integrated using 32 Euler steps, progressively transporting the particles toward likely image locations.

This produces a distribution of candidate locations rather than a single prediction. The final geolocation estimate is obtained from the mode of the particle distribution.
