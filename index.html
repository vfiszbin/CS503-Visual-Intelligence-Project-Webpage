CS503 Visual Intelligence - Final Project

Flow Matching for Probabilistic Image Geolocalization in Switzerland

Vincent Fiszbin, Ziyi Liu, Yuchen Ouyang, Yu Chenxu
Final Slides Progress Report Proposal Website Code

Given a Swiss street-view image, predict both a location estimate and a spatial probability distribution.

Abstract

We study image geolocalization in Switzerland: estimating where a street-view image was captured from visual cues alone. Instead of returning only one coordinate, the project explores probabilistic flow-matching models that produce spatial distributions over possible locations.

The final system compares a regression baseline, a spherical Riemannian flow-matching baseline, a local Euclidean flow-matching model on Switzerland, and a tuned R2 model conditioned with classifier priors.

The strongest model reaches a 13.3 km mode median distance and 58.96% canton accuracy. The final experiments show that text/caption conditioning did not help, while matching the geometry to the Switzerland-only task and adding regional priors gave the clearest gains.
